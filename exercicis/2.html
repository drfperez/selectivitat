Càlculs de poder calorífic, calor i cabal de combustible

Un motor de combustió interna de cilindrada 2.000cc, en el qual s'està utilitzant gasoil com a combustible, necessita una potència de 60 kW per a la seva correcte funcionament. Si la calor específica del gasoil és de 42,4 kJ/kg i el seu poder calorífic superior és de 43,5 MJ/kg, quin cabal de gasoil s'ha de fer arribar al motor cada minut si el rendiment del motor és del 30%?

Aquest codi concret és un exemple d'una pàgina web HTML que realitza càlculs relacionats amb un motor de combustió interna i el seu consum de gasoil. Aquí tens una explicació línia per línia:

  • <!DOCTYPE html>: Aquesta línia declara la versió de HTML que s'utilitza, que és HTML5.
  • <html>: Inicia l'element HTML.
  • <head>: Inicia la capçalera de la pàgina, que conté metainformació i vincles a altres arxius com CSS i JavaScript.
  • Metainformació: Les dues línies <meta> especifiquen la codificació de caràcters i l'escala inicial per a dispositius mòbils.
  • <title>: Defineix el títol de la pàgina que es mostrarà a la barra de navegació del navegador.
  • <script>: Importa el JavaScript de Bootstrap per a la funcionalitat de la pàgina.
  • <link>: Importa l'arxiu CSS de Bootstrap per a l'estil de la pàgina.
  • </head>: Tanca la capçalera.
  • <body>: Inicia el cos de la pàgina web.
  • <div id="navbarContainer"></div>: Crea un contenidor buit amb un identificador on es carregarà un contingut des d'un arxiu extern mitjançant JavaScript.
  • <script>: Inicia una secció de codi JavaScript que realitza una sol·licitud AJAX per obtenir contingut de "navbar.html" i el posa dins del div amb l'ID "navbarContainer".
  • <h1>: Mostra un títol principal a la pàgina.
  • <p>: Mostra un paràgraf amb una pregunta sobre el càlcul de consum de gasoil en un motor.
  • <form>: Inicia un formulari amb diferents camps d'entrada per a l'usuari i un botó per a calcular.
  • <script>: Inicia una funció JavaScript anomenada "calcular" que realitza els càlculs i mostra el resultat a la pàgina.
  • <script src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js"></script>: Importa el script de MathJax per a la representació de notació matemàtica a la pàgina.
  • <script>: Inicia una funció JavaScript anomenada "calcular" que realitza els càlculs i mostra el resultat a la pàgina.
  • <!-- Obtenir les dades que ha introduït l'usuari -->: Comentari que indica l'inici de la captura de dades del formulari.
  • var potencia = document.getElementById("potencia").value;: Obté el valor introduït a l'element amb l'ID "potencia" i l'emmagatzema en la variable "potencia".
  • var calor_especifica = document.getElementById("calor_especifica").value;: Obté el valor introduït a l'element amb l'ID "calor_especifica" i l'emmagatzema en la variable "calor_especifica".
  • var poder_calorifico_superior = document.getElementById("poder_calorifico_superior").value;: Obté el valor introduït a l'element amb l'ID "poder_calorifico_superior" i l'emmagatzema en la variable "poder_calorifico_superior".
  • var rendimiento = document.getElementById("rendimiento").value/100;: Obté el valor introduït a l'element amb l'ID "rendimiento" i el converteix a una fracció dividint-lo per 100, i l'emmagatzema en la variable "rendimiento".
  • // Calcular la quantitat de calor que es produeix cada minut: Comentari que explica l'operació següent.
  • var calor = potencia * 60;: Calcula la quantitat de calor produïda cada minut multiplicant la potència per 60 i emmagatzema el resultat en la variable "calor".
  • // Calcular la massa de gasoil necessària per produir aquesta quantitat de calor: Comentari que explica l'operació següent.
  • var massa_gasoil = calor / (calor_especifica * poder_calorifico_superior);: Calcula la massa de gasoil necessària per produir la quantitat de calor utilitzant les dades introduïdes i emmagatzema el resultat en la variable "massa_gasoil".
